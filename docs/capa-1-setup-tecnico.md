# Capa 1 - Setup técnico inicial

La Capa 1 prepara la estructura técnica mínima del proyecto.

## Objetivo

Dejar AuraNails listo para ejecutar backend, frontend y base de datos en entorno local.

## Estructura objetivo

```text
AuraNails/
├── backend/
├── frontend/
├── docs/
└── docker-compose.yml
```

## Backend previsto

- Django.
- Django REST Framework.
- PostgreSQL.
- JWT en una capa posterior.

## Frontend previsto

- React.
- Vite.
- Tailwind CSS.
- React Router.
- Axios.

## Estado

En preparación.

## Criterio de cierre

La capa se considera terminada cuando:

- El backend levanta localmente.
- El frontend levanta localmente.
- PostgreSQL queda disponible.
- El README explica cómo ejecutar el proyecto.
- No hay módulos de negocio todavía.
