# Visión de AuraNails

AuraNails busca ser una aplicación simple, visual y práctica para manicuras y pequeños estudios de uñas.

El proyecto nace de una necesidad concreta: muchas profesionales gestionan sus turnos, clientas, pagos, señas y productos mediante WhatsApp, libretas, notas del celular o planillas sueltas. Eso funciona al principio, pero se vuelve difícil de sostener cuando el trabajo crece.

AuraNails propone centralizar esa información en una herramienta pensada para el día a día real de una manicura.

## Problema que resuelve

Una manicura necesita organizar varias cosas al mismo tiempo:

- Turnos.
- Clientas.
- Servicios.
- Señas.
- Cobros.
- Productos.
- Gastos.
- Fotos de trabajos.
- Recordatorios.

El problema no es solo guardar datos. El verdadero problema es poder entender rápidamente qué tiene que hacer, cuánto cobró, qué falta confirmar y qué productos necesita reponer.

## Principio principal

La aplicación debe ser fácil de usar para una persona con poco o nulo conocimiento técnico.

Esto significa que AuraNails debe evitar interfaces cargadas, lenguaje administrativo y flujos innecesariamente largos.

La app debe sentirse como una herramienta de trabajo cotidiana, no como un sistema empresarial.

## Usuario principal

El usuario principal es una manicura independiente o dueña de un pequeño estudio.

Características esperadas:

- Usa el celular como herramienta principal.
- Organiza gran parte del trabajo por WhatsApp.
- Necesita ver sus turnos rápido.
- No quiere cargar datos complejos.
- Quiere saber cuánto ganó sin hacer cuentas manuales.
- Necesita recordar preferencias de cada clienta.
- Valora mucho las fotos de trabajos realizados.

## Promesa del producto

AuraNails debe ayudar a que una manicura pueda decir:

> Tengo mis turnos, clientas, cobros, productos y trabajos anteriores en un solo lugar.

## Qué debe sentirse simple

- Crear un turno.
- Confirmar una seña.
- Cobrar un servicio.
- Ver qué clienta sigue.
- Ver qué producto falta.
- Revisar trabajos anteriores.
- Saber cuánto ingresó en el mes.

## Qué no debe ser AuraNails

AuraNails no debe convertirse en un ERP complejo, un sistema contable avanzado ni una plataforma cargada de funciones difíciles de entender.

No se busca cubrir todo desde el primer día.

Se busca construir una base sólida, útil y escalable.
